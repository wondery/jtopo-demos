Node

Node

节点对象，形状为矩形； 核心属性：坐标(x,y)和尺寸(width, height)

创建一个节点, 通常有以下几种形式：

var node = new Node();

var node = new Node('name');

var node = new Node('name', 10, 10);

var node = new Node('name', 10, 10, 30, 30);

构造器

new Node(text, x, y, width, height)

所有参数可以为空，通过其他方法按需设置

参数:
Name Type Description
text string

文本
x x

坐标x
y y

坐标y
width number

宽度
height number

高度

父类:

  • DisplayObject

属性

children :Array

继承自:
  • DisplayObject#children

子节点集合

connectable :Boolean

继承自:
  • DisplayObject#connectable

是否可连线，仅仅编辑模式有意义

deep :Number

继承自:
  • DisplayObject#deep

层级深度, Layer为0，下面第一层子节点为1，依次类推

depth :Number

继承自:
  • DisplayObject#depth

厚度（仅3D视图下有意义）

draggable :Boolean

继承自:
  • DisplayObject#draggable

是否可拖拽

editable :Boolean

继承自:
  • DisplayObject#editable

是否可以调整角度、尺寸，仅仅编辑模式有意义

height :Number

继承自:
  • DisplayObject#height

高度

id

继承自:
  • DisplayObject#id

唯一标志，不能有重复

imageSrc :String

图片的src

inLinks :Array

所有以本节点为结束点的Link对象数组

isNode :Boolean

是否是Node类的对象, 主要用于和Link区分

mouseEnabled :Boolean

继承自:
  • DisplayObject#mouseEnabled

是否响应鼠标，为false的时候，不响应鼠标的动作

origin :Array

继承自:
  • DisplayObject#origin

本地坐标系的原点，默认为[0,0], 父节点的左上角

originAutoRotate :Boolean

继承自:
  • DisplayObject#originAutoRotate

是否跟随父节点(当父节点是Link的时候)旋转

outLinks :Array

所有以本节点为开始点的Link对象数组

parent :Object

继承自:
  • DisplayObject#parent

上级对象

rotation :Number

继承自:
  • DisplayObject#rotation

旋转角度

scaleX :Number

继承自:
  • DisplayObject#scaleX

水平缩放系数

scaleY :Number

继承自:
  • DisplayObject#scaleY

垂直缩放系数

selectedStyle :Object

继承自:
  • DisplayObject#selectedStyle

选中时的样式，如果不设置将使用默认的

serializeable :Boolean

继承自:
  • DisplayObject#serializeable

是否可序列化，默认true 如果为false，序列化时将跳过该对象

serializers

序列化属性列表

showSelected :Boolean

继承自:
  • DisplayObject#showSelected

是否显示选中的效果，默认显示

style :Style

继承自:
  • DisplayObject#style

样式

text :String|null

节点文本

多行直接可以使用\n换行

textOffsetX :Number

文本水平偏移量

textOffsetY :Number

文本垂直偏移量

userData :Object

继承自:
  • DisplayObject#userData

为了安全考虑，防止与对象属性冲突,用户最好使用该对象 用户附加到该对象的自定义属性

visible :Boolean

继承自:
  • DisplayObject#visible

是否可见

width :Number

继承自:
  • DisplayObject#width

宽度

x :Number

继承自:
  • DisplayObject#x

坐标x

y :Number

继承自:
  • DisplayObject#y

坐标y

zIndex :Number

继承自:
  • DisplayObject#zIndex

zIndex显示前后排序

Node默认为2，Link默认为1, Link默认绘制在Node的后面

方法

addChild(child)

继承自:
  • DisplayObject#addChild

增加子对象

参数:
Name Type Description
child Node

addChilds(child)

继承自:
  • DisplayObject#addChilds

一次性增加多个对象, 比单个增加速度快多，数量越多越明显！

参数:
Name Type Description
child DisplayObject

addEventListener(type, callback)

继承自:
  • DisplayObject#addEventListener

增加事件监听

参数:
Name Type Description
type String

事件类型
callback function

css(name, value)

继承自:
  • DisplayObject#css

1.3.8 新增方法，语法上支持一部分css样式

可以每次设置一个样式，如：obj.css('border', '1px solid ');

也可以一次设置一组样式，如: 

node.css({
    width: 48,
    height: '48px',
